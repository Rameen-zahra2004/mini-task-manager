📌 Mini Task Manager (MERN Stack)
🚀 Overview

Mini Task Manager is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

It allows users to create, update, delete, and manage tasks efficiently with a clean and responsive UI.

This project demonstrates real-world full-stack development skills including REST API design, frontend-backend integration, and scalable project architecture.

✨ Features
➕ Create new tasks
✏️ Update existing tasks
❌ Delete tasks
📋 View all tasks in organized layout
🔍 Filter tasks by status/priority
⚡ Responsive UI (React + Vite)
🔗 REST API integration (Frontend ↔ Backend)

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/Rameen-zahra2004/mini-task-manager.git
cd mini-task-manager
2️⃣ Setup Backend
cd server
npm install

Create .env file:

MONGO_URI=your_mongodb_connection_string
PORT=5000

Run backend:

npm start
3️⃣ Setup Frontend
cd client
npm install
npm run dev

Frontend runs at:

http://localhost:5173
🔗 API Integration

Frontend communicates with backend using Axios.

Base API URL:

http://localhost:5000/api/tasks
